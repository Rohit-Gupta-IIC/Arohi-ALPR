# Arohi-ALPR
This is a application build using OpenALPR and Python

### Dependencies:
1. OpenALPR
2. Python3.6
3. Tkinter
4. SQLite3

### How to execute
```
sudo chmod 777 setup.sh
python3 logic.cypython-36.pyc

```

### The Output can be found out by the following commands:
```
sqlitebrowser Aarohi.db
```
